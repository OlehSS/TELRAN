1. Перед началом работы нужно настоить подпись на Git Bush. На GitHub: Перейти по ссылке на адрес репозитория -> Code -> SSH(копировать ссылку) -> перейти в рабочую директорию -> выполнить команду git clone "ссылка на GitHub". 

2. Для того чтобы получить изменения с github.com на локальный компьютер нужно использовать команду git pull.

3. git status - показывает состояние файлов в нашем репозитории (Working Directory, Staging Area, .git directory). git log - показывает историю коммитов.
